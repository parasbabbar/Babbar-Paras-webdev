Code Over-Flow

Problem Statement: Code Over-flow is a single page Mean Stack application that uses Stack Overflow api where programmers can learn, share their knowledge and advance their careers.

Domain objects: Question, Answers, Follow-up Discussions

A Question can have several answers and these even leads to further follow-up queries or discussions

Human users: Admin,Programmer1(That Ask the Question), Programmer2(Who Answers the Question) & Anonymous user

Admin: CRUD Problems,Solutions and Users

Anonymous user: Search/view the Problems from the existing Solutions

Programmer-1: Same as anonymous user, But He can raise questions and give ratings to the Programmer-2 according to the help given by them and even can share his knowledge as well by posting some useful article or links.

Programmer-2: Same as Programmer-1, But He is the one who answers questions.
